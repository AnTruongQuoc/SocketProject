# SocketProject

Đây là đồ án cho môn học Mạng máy tính (Computer Network)

## Mô tả

Đồ án này thực hiện việc xây dựng một protocol (sử dụng giao thức TCP/UDP) kết nối giữa Client và Server.

Xây dựng Console UI cho Client và Server.

### Chức năng

Chức năng kết nối từ Client đến Server.

Chức năng đăng nhập và quản lý thông tin người dùng.

Chức năng upload, download dữ liệu.

Chức năng chat với người dùng khác (tạo group chat nhiều người kết nối).
